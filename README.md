# Warehouse Management API

RESTful API сервис для автоматизации складского учета. Разработан как готовое B2B-решение для реального складского комплекса. Обеспечивает строгий контроль остатков, аудит действий персонала и безопасную авторизацию.

## Ключевые возможности (Features)
* **Ролевая модель доступа (RBAC):** Разделение прав на 'admin' (управление номенклатурой, просмотр логов) и 'worker' (только изменение остатков).
* **Транзакционная целостность:** Защита от отрицательных остатков на складе при одновременных списаниях.
* **Система аудита:** Автоматическое логирование всех операций с товарами (кто, когда и сколько списал/добавил).
* **Безопасность:** JWT-авторизация и хэширование паролей (bcrypt).
* **Автодокументирование:** Встроенный Swagger UI / Redoc.

## Технологический стек
* **Backend:** Python 3, FastAPI
* **Database:** SQLite3 (SQLAlchemy ORM)
* **Auth:** JWT, Passlib
* **Deploy:** Docker, Docker Compose

## Схема базы данных
* 'users' - Учетные данные и роли сотрудников.
* 'items' - Справочник номенклатуры и текущие остатки.
* 'item_logs' - Журнал истории движения товаров (связан с 'users' и 'items').
